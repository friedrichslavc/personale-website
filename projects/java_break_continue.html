Java Break 和 Continue 语句

Break 和 Continue 是 Java 中用于控制循环流程的两个重要语句。它们可以改变循环的正常执行顺序，提供更灵活的控制结构。

1. Break 语句

Break 语句用于跳出循环或 switch 语句。当遇到 break 时，程序会立即退出最内层的循环或 switch 语句。

1.1 在 for 循环中使用 break


for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;
    }
    System.out.println(i);
}
// 输出: 0 1 2 3 4

1.2 在 while 循环中使用 break


int i = 0;
while (true) {
    if (i == 5) {
        break;
    }
    System.out.println(i);
    i++;
}
// 输出: 0 1 2 3 4

2. Continue 语句

Continue 语句用于跳过循环的当前迭代，直接进入下一次迭代。

2.1 在 for 循环中使用 continue


for (int i = 0; i < 5; i++) {
    if (i == 2) {
        continue;
    }
    System.out.println(i);
}
// 输出: 0 1 3 4

2.2 在 while 循环中使用 continue


int i = 0;
while (i < 5) {
    i++;
    if (i == 3) {
        continue;
    }
    System.out.println(i);
}
// 输出: 1 2 4 5

3. 在嵌套循环中使用 break 和 continue

在嵌套循环中，break 和 continue 默认只影响最内层循环。如果需要影响外层循环，可以使用标签。


outerLoop: for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        if (i == 1 && j == 1) {
            break outerLoop;
        }
        System.out.println("i = " + i + ", j = " + j);
    }
}

4. 注意事项

  • 过度使用 break 和 continue 可能会使代码难以理解和维护。
  • 在使用 break 和 continue 时，要确保不会意外跳过重要的代码逻辑。
  • 在嵌套循环中使用标签时要格外小心，确保跳转到正确的循环层级。
