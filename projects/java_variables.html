Java 变量

变量是用于存储数据值的容器。在Java中，每个变量都必须有一个特定的数据类型。

1. 变量声明

在Java中，变量声明的基本语法是：


type variableName = value;

例如：


int myNum = 5;               // 整数
float myFloatNum = 5.99f;    // 浮点数
char myLetter = 'D';         // 字符
boolean myBool = true;       // 布尔
String myText = "Hello";     // 字符串

2. 变量类型

基本数据类型：

  • byte: 8位有符号二进制补码整数
  • short: 16位有符号二进制补码整数
  • int: 32位有符号二进制补码整数
  • long: 64位有符号二进制补码整数
  • float: 单精度32位IEEE 754浮点数
  • double: 双精度64位IEEE 754浮点数
  • boolean: true 或 false
  • char: 单个16位Unicode字符

引用数据类型：

  • 类对象
  • 数组
  • 接口

3. 变量命名规则

  • 变量名可以包含字母、数字、下划线和美元符号
  • 变量名必须以字母、下划线或美元符号开头
  • 变量名区分大小写
  • 变量名不能是Java关键字
  • 变量名应该有描述性，遵循驼峰命名法（如 myVariableName）

4. 变量作用域

变量的作用域决定了变量在程序中的可见性和生命周期。

局部变量：

在方法或代码块内部声明的变量。


public void myMethod() {
    int x = 10;  // 局部变量
    // x 只在 myMethod 内部可用
}

实例变量（非静态字段）：

在类中声明，但在方法外声明的变量。


public class MyClass {
    int y = 5;  // 实例变量
    // y 可以被这个类的所有非静态方法访问
}

类变量（静态字段）：

使用 static 关键字声明的变量，被类的所有实例共享。


public class MyClass {
    static int z = 0;  // 类变量
    // z 被所有 MyClass 的实例共享
}

5. 常量

常量是不能被修改的变量，在Java中使用 final 关键字声明。


final double PI = 3.14159;

6. 类型推断（Java 10+）

从Java 10开始，可以使用 var 关键字进行局部变量类型推断。


var message = "Hello, Java!";  // 推断为 String 类型
var number = 42;               // 推断为 int 类型

7. 变量初始化

局部变量在使用前必须初始化，而实例变量和类变量有默认值。


public class VariableInitialization {
    int instanceVar;  // 默认值为 0
    static boolean classVar;  // 默认值为 false

    public void method() {
        int localVar;
        // System.out.println(localVar);  // 错误：localVar 未初始化
        localVar = 10;  // 正确：初始化后可以使用
        System.out.println(localVar);
    }
}

注意事项

  • 选择有意义的变量名可以提高代码的可读性。
  • 尽量将变量的作用域限制在最小范围内。
  • 对于不会改变的值，使用常量（final 变量）。
  • 注意变量的生命周期，及时释放不再使用的资源。
  • 使用适当的数据类型可以优化内存使用和程序性能。
