Java While 循环

While 循环是 Java 中的一种基本循环结构，用于在满足特定条件时重复执行一段代码。

1. While 循环基本语法


while (condition) {
    // 循环体
    // 当 condition 为 true 时，重复执行这里的代码
}

循环会一直执行，直到条件变为 false。

2. While 循环示例


int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}

这个例子会输出 0 到 4 的数字。

3. 无限循环

如果条件始终为 true，while 循环将无限执行。虽然有时这是有意为之，但通常是由于编程错误导致的。


while (true) {
    // 这个循环将永远执行
    // 通常需要一个 break 语句来退出
}

4. Do-While 循环

Do-While 循环是 While 循环的变体，它保证循环体至少执行一次。


do {
    // 循环体
} while (condition);

示例：


int i = 0;
do {
    System.out.println(i);
    i++;
} while (i < 5);

5. While 循环与 If 语句结合


int i = 0;
while (i < 10) {
    if (i % 2 == 0) {
        System.out.println(i + " 是偶数");
    }
    i++;
}

6. 使用 break 语句

break 语句可以用来提前退出循环。


int i = 0;
while (true) {
    if (i >= 5) {
        break;  // 当 i 大于或等于 5 时退出循环
    }
    System.out.println(i);
    i++;
}

7. 使用 continue 语句

continue 语句用于跳过当前迭代的剩余部分，直接进入下一次迭代。


int i = 0;
while (i < 10) {
    i++;
    if (i % 2 == 0) {
        continue;  // 跳过偶数
    }
    System.out.println(i);  // 只打印奇数
}

8. 嵌套 While 循环


int i = 1;
while (i <= 3) {
    int j = 1;
    while (j <= 3) {
        System.out.println(i + "," + j);
        j++;
    }
    i++;
}

9. While 循环与数组


String[] fruits = {"Apple", "Banana", "Cherry"};
int index = 0;
while (index < fruits.length) {
    System.out.println(fruits[index]);
    index++;
}

10. 实际应用示例：猜数字游戏


import java.util.Random;
import java.util.Scanner;

public class GuessNumberGame {
    public static void main(String[] args) {
        Random random = new Random();
        int numberToGuess = random.nextInt(100) + 1;
        Scanner scanner = new Scanner(System.in);
        boolean hasGuessedCorrectly = false;

        System.out.println("我想了一个1到100之间的数。你能猜出来吗？");

        while (!hasGuessedCorrectly) {
            System.out.print("请输入你的猜测：");
            int guess = scanner.nextInt();

            if (guess < numberToGuess) {
                System.out.println("太小了！再试一次。");
            } else if (guess > numberToGuess) {
                System.out.println("太大了！再试一次。");
            } else {
                hasGuessedCorrectly = true;
                System.out.println("恭喜你，猜对了！");
            }
        }

        scanner.close();
    }
}

注意事项

  • 确保循环条件最终会变为 false，否则会导致无限循环。
  • 在循环内更新控制变量，以确保循环最终会结束。
  • 对于已知迭代次数的循环，考虑使用 for 循环代替 while 循环。
  • 使用 while 循环时要小心，确保不会意外创建无限循环。
  • 在处理用户输入或文件读取等情况时，while 循环特别有用。
