Java 语法

1. 基本语法规则

  • Java 程序的基本单位是类（class）
  • 每个 Java 程序都必须有一个 main 方法，这是程序的入口点
  • Java 使用花括号 {} 来定义代码块
  • 每个语句都以分号 ; 结束
  • Java 区分大小写

2. Java 程序结构


// 包声明（可选）
package com.example;

// 导入语句（可选）
import java.util.Scanner;

// 类定义
public class MyProgram {
    // 主方法
    public static void main(String[] args) {
        // 程序代码
    }

    // 其他方法
    public void someMethod() {
        // 方法代码
    }
}

3. 变量声明和初始化


// 变量声明
int number;
String name;

// 变量初始化
int age = 25;
String message = "Hello, Java!";

// 常量声明（使用 final 关键字）
final double PI = 3.14159;

4. 数据类型

基本数据类型：

  • byte, short, int, long（整数类型）
  • float, double（浮点数类型）
  • boolean（布尔类型）
  • char（字符类型）

引用数据类型：

  • 类（Class）
  • 接口（Interface）
  • 数组（Array）

5. 运算符

  • 算术运算符：+, -, *, /, %, ++, --
  • 关系运算符：==, !=, >, <, >=, <=
  • 逻辑运算符：&&, ||, !
  • 赋值运算符：=, +=, -=, *=, /=, %=
  • 位运算符：&, |, ^, ~, <<, >>, >>>

6. 控制流语句

if-else 语句：


if (condition) {
    // 代码块
} else if (anotherCondition) {
    // 代码块
} else {
    // 代码块
}

switch 语句：


switch (variable) {
    case value1:
        // 代码块
        break;
    case value2:
        // 代码块
        break;
    default:
        // 代码块
}

循环语句：


// for 循环
for (int i = 0; i < 5; i++) {
    // 代码块
}

// while 循环
while (condition) {
    // 代码块
}

// do-while 循环
do {
    // 代码块
} while (condition);

7. 方法定义


public static int add(int a, int b) {
    return a + b;
}

8. 类和对象


public class Car {
    // 属性
    private String brand;
    private int year;

    // 构造方法
    public Car(String brand, int year) {
        this.brand = brand;
        this.year = year;
    }

    // 方法
    public void start() {
        System.out.println("The car is starting.");
    }
}

// 创建对象
Car myCar = new Car("Toyota", 2022);
myCar.start();

10. 包和导入


package com.example.myproject;

import java.util.ArrayList;
import java.util.List;

注意事项

  • Java 源文件的名称必须与公共类的名称相同，并以 .java 为扩展名
  • 一个源文件中只能有一个公共类
  • 注意缩进，保持代码的可读性
  • 遵循 Java 命名约定可以提高代码的可读性和维护性
