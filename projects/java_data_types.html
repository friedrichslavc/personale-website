Java 数据类型

Java 是一种强类型语言，这意味着每个变量和表达式都有一个类型。Java 中的数据类型分为两大类：基本数据类型和引用数据类型。

1. 基本数据类型

Java 有 8 种基本数据类型：

1.1 整数类型

  • byte: 8 位有符号二进制补码整数
    • 最小值：-128 (-2^7)
    • 最大值：127 (2^7 - 1)
    • 默认值：0
  • short: 16 位有符号二进制补码整数
    • 最小值：-32,768 (-2^15)
    • 最大值：32,767 (2^15 - 1)
    • 默认值：0
  • int: 32 位有符号二进制补码整数
    • 最小值：-2,147,483,648 (-2^31)
    • 最大值：2,147,483,647 (2^31 - 1)
    • 默认值：0
  • long: 64 位有符号二进制补码整数
    • 最小值：-9,223,372,036,854,775,808 (-2^63)
    • 最大值：9,223,372,036,854,775,807 (2^63 - 1)
    • 默认值：0L

1.2 浮点类型

  • float: 单精度 32 位 IEEE 754 浮点数
    • 默认值：0.0f
    • 范例：3.14f
  • double: 双精度 64 位 IEEE 754 浮点数
    • 默认值：0.0d
    • 范例：3.14159265359

1.3 字符类型

  • char: 单个 16 位 Unicode 字符
    • 最小值：'\u0000'（即为 0）
    • 最大值：'\uffff'（即为 65,535）
    • 默认值：'\u0000'

1.4 布尔类型

  • boolean: 表示一位的信息
    • 只有两个取值：true 和 false
    • 默认值：false

2. 引用数据类型

引用数据类型指向一个对象，并用来访问对象的方法和变量。主要包括：

  • 类（Class）
  • 接口（Interface）
  • 数组（Array）

2.1 String 类

虽然 String 是一个类，但因为使用频繁，Java 对其进行了特殊处理。


String greeting = "Hello, World!";

2.2 数组

数组是一种用于存储多个相同类型数据的容器。


int[] numbers = {1, 2, 3, 4, 5};
String[] names = {"Alice", "Bob", "Charlie"};

3. 类型转换

3.1 自动类型转换（隐式转换）

从小类型到大类型的转换是自动的：


byte -> short -> int -> long -> float -> double
char -> int

3.2 强制类型转换（显式转换）

从大类型到小类型需要显式转换：


int i = 100;
byte b = (byte)i;

4. 包装类

每个基本数据类型都有对应的包装类：

  • Byte
  • Short
  • Integer
  • Long
  • Float
  • Double
  • Character
  • Boolean

包装类提供了许多有用的方法，如类型转换和值比较。

5. 示例


// 基本数据类型
int age = 25;
double salary = 5000.50;
char grade = 'A';
boolean isStudent = true;

// 引用数据类型
String name = "John Doe";
int[] scores = {85, 90, 78, 88, 76};

// 包装类
Integer wrappedAge = Integer.valueOf(age);
Double wrappedSalary = salary;  // 自动装箱

// 类型转换
long longValue = age;  // 自动类型转换
int intValue = (int)salary;  // 强制类型转换

注意事项

  • 选择合适的数据类型可以优化内存使用和提高程序性能。
  • 在进行类型转换时要注意可能的数据丢失或精度损失。
  • 使用包装类时要注意自动装箱和拆箱的性能影响。
  • 在处理浮点数时，要注意精度问题，特别是在金融计算中。
