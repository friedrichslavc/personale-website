Java For 循环

For 循环是 Java 中最常用的循环结构之一，特别适用于需要执行特定次数的循环。它提供了一种简洁的方式来迭代一定范围的值或集合中的元素。

1. 基本 For 循环语法


for (初始化; 条件; 更新) {
    // 循环体
}
  • 初始化： 在循环开始前执行一次，通常用于初始化循环计数器。
  • 条件： 在每次循环迭代开始时检查，如果为 true 则执行循环体。
  • 更新： 在每次循环迭代结束时执行，通常用于更新循环计数器。

2. 基本 For 循环示例


for (int i = 0; i < 5; i++) {
    System.out.println("当前计数: " + i);
}

3. For 循环变体

3.1 多个初始化或更新语句


for (int i = 0, j = 10; i < j; i++, j--) {
    System.out.println("i = " + i + ", j = " + j);
}

3.2 无限 For 循环


for (;;) {
    // 这是一个无限循环
    // 需要使用 break 语句来退出
}

4. 增强型 For 循环（For-Each 循环）

用于遍历数组或集合中的元素，语法更简洁。


String[] fruits = {"Apple", "Banana", "Cherry"};
for (String fruit : fruits) {
    System.out.println(fruit);
}

5. 嵌套 For 循环


for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        System.out.println("i = " + i + ", j = " + j);
    }
}

6. 使用 break 和 continue

6.1 使用 break 提前退出循环


for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break; // 当 i 等于 5 时退出循环
    }
    System.out.println(i);
}

6.2 使用 continue 跳过当前迭代


for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) {
        continue; // 跳过偶数
    }
    System.out.println(i); // 只打印奇数
}

7. 标签化 For 循环

可以给循环添加标签，用于在嵌套循环中跳出外层循环。


outerLoop: for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        if (i == 1 && j == 1) {
            break outerLoop; // 跳出外层循环
        }
        System.out.println("i = " + i + ", j = " + j);
    }
}

8. For 循环与数组


int[] numbers = {1, 2, 3, 4, 5};
for (int i = 0; i < numbers.length; i++) {
    System.out.println("索引 " + i + " 的值是: " + numbers[i]);
}

9. For 循环与 ArrayList


import java.util.ArrayList;

ArrayList cars = new ArrayList();
cars.add("Volvo");
cars.add("BMW");
cars.add("Ford");

for (int i = 0; i < cars.size(); i++) {
    System.out.println(cars.get(i));
}

10. 实际应用示例：打印乘法表


public class MultiplicationTable {
    public static void main(String[] args) {
        for (int i = 1; i <= 9; i++) {
            for (int j = 1; j <= i; j++) {
                System.out.printf("%d x %d = %d\t", j, i, i * j);
            }
            System.out.println(); // 换行
        }
    }
}

注意事项

  • 确保循环有明确的终止条件，避免无限循环。
  • 注意循环变量的作用域，for 循环中声明的变量只在循环内部可见。
  • 对于简单的数组或集合遍历，优先使用增强型 for 循环（for-each）。
  • 在循环体中修改循环变量可能导致意外行为，应谨慎使用。
  • 大型循环中的性能考虑：尽量将不变的计算移到循环外。
