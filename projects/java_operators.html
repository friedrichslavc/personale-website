Java 运算符

Java 提供了多种运算符用于执行不同类型的操作。这些运算符可以分为几个主要类别：

1. 算术运算符

用于执行基本的数学运算。

运算符描述示例
+加法a + b
-减法a - b
*乘法a * b
/除法a / b
%取模（余数）a % b
++递增a++ 或 ++a
--递减a-- 或 --a

2. 关系运算符

用于比较两个值。

运算符描述示例
==等于a == b
!=不等于a != b
>大于a > b
<小于a < b
>=大于或等于a >= b
<=小于或等于a <= b

3. 逻辑运算符

用于组合布尔表达式。

运算符描述示例
&&逻辑与a && b
||逻辑或a || b
!逻辑非!a

4. 赋值运算符

用于给变量赋值。

运算符描述示例
=简单赋值a = 5
+=加法赋值a += 3 等价于 a = a + 3
-=减法赋值a -= 3 等价于 a = a - 3
*=乘法赋值a *= 3 等价于 a = a * 3
/=除法赋值a /= 3 等价于 a = a / 3
%=取模赋值a %= 3 等价于 a = a % 3

5. 位运算符

用于操作二进制位。

运算符描述示例
&按位与a & b
|按位或a | b
^按位异或a ^ b
~按位取反~a
<<左移a << 2
>>右移a >> 2
>>>无符号右移a >>> 2

6. 三元运算符

条件运算符，也称为三元运算符。


variable = (condition) ? expressionTrue : expressionFalse;

7. instanceof 运算符

用于检查对象是否是特定类型（类或接口）的实例。


object instanceof Type

8. 运算符优先级

运算符的优先级决定了表达式中运算执行的顺序。以下是从高到低的优先级顺序：

  1. 后缀运算符：() [] . (点运算符)
  2. 一元运算符：++ -- ! ~
  3. 乘性运算符：* / %
  4. 加性运算符：+ -
  5. 移位运算符：<< >> >>>
  6. 关系运算符：< <= > >= instanceof
  7. 相等运算符：== !=
  8. 按位与运算符：&
  9. 按位异或运算符：^
  10. 按位或运算符：|
  11. 逻辑与运算符：&&
  12. 逻辑或运算符：||
  13. 条件运算符：?:
  14. 赋值运算符：= += -= *= /= %= &= ^= |= <<= >>= >>>=

9. 示例


public class OperatorExample {
    public static void main(String[] args) {
        // 算术运算符
        int a = 10, b = 20;
        System.out.println("a + b = " + (a + b));
        System.out.println("a - b = " + (a - b));
        System.out.println("a * b = " + (a * b));
        System.out.println("b / a = " + (b / a));
        System.out.println("b % a = " + (b % a));
        System.out.println("a++ = " + (a++));
        System.out.println("++a = " + (++a));

        // 关系运算符
        System.out.println("a == b = " + (a == b));
        System.out.println("a != b = " + (a != b));
        System.out.println("a > b = " + (a > b));
        System.out.println("a < b = " + (a < b));

        // 逻辑运算符
        boolean x = true, y = false;
        System.out.println("x && y = " + (x && y));
        System.out.println("x || y = " + (x || y));
        System.out.println("!x = " + (!x));

        // 位运算符
        int c = 60, d = 13;
        System.out.println("c & d = " + (c & d));
        System.out.println("c | d = " + (c | d));
        System.out.println("c ^ d = " + (c ^ d));
        System.out.println("~c = " + (~c));
        System.out.println("c << 2 = " + (c << 2));
        System.out.println("c >> 2 = " + (c >> 2));

        // 三元运算符
        int result = (a > b) ? a : b;
        System.out.println("Result of ternary operator: " + result);

        // instanceof 运算符
        String str = "Hello";
        System.out.println("str instanceof String = " + (str instanceof String));
    }
}

注意事项

  • 在使用除法运算符时，要注意整数除法和浮点数除法的区别。
  • 使用位运算符时要小心，确保你理解它们的作用。
  • 逻辑运算符（&& 和 ||）具有短路效应，这在某些情况下可能会影响程序的行为。
  • 使用复合赋值运算符（如 +=）可以使代码更简洁，但要确保它不会降低代码的可读性。
  • 在复杂表达式中，建议使用括号来明确运算顺序，提高代码的可读性。
