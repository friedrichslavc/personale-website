Java 输出

在Java中，有多种方法可以输出数据。最常用的是通过 System.out 对象来实现。

1. println() 方法

println() 方法用于输出内容并在末尾添加换行符。


System.out.println("Hello, World!");
System.out.println("This is a new line.");

输出结果：

Hello, World!
This is a new line.

2. print() 方法

print() 方法用于输出内容，但不会在末尾添加换行符。


System.out.print("Hello, ");
System.out.print("World!");

输出结果：

Hello, World!

3. printf() 方法

printf() 方法允许格式化输出，类似于C语言的 printf 函数。


String name = "Alice";
int age = 25;
System.out.printf("Name: %s, Age: %d%n", name, age);

输出结果：

Name: Alice, Age: 25

常用的格式说明符：

  • %s - 字符串
  • %d - 整数
  • %f - 浮点数
  • %c - 字符
  • %b - 布尔值
  • %n - 换行符（与平台无关）

4. 输出变量

可以直接输出变量的值：


int number = 42;
System.out.println(number);  // 输出: 42

String message = "Hello, Java!";
System.out.println(message);  // 输出: Hello, Java!

5. 连接字符串和变量

使用 + 运算符可以连接字符串和变量：


String name = "John";
int age = 30;
System.out.println("My name is " + name + " and I am " + age + " years old.");

输出结果：

My name is John and I am 30 years old.

6. 输出特殊字符

使用转义序列可以输出特殊字符：


System.out.println("This is a tab:\t and this is a new line:\n");
System.out.println("This is a \"quoted\" text");

输出结果：

This is a tab:	 and this is a new line:

This is a "quoted" text

7. 输出到错误流

使用 System.err 可以输出到标准错误流：


System.err.println("This is an error message");

8. 使用 StringBuilder 构建大型输出

对于需要构建大量输出的情况，使用 StringBuilder 可以提高效率：


StringBuilder sb = new StringBuilder();
for (int i = 0; i < 5; i++) {
    sb.append("Line ").append(i + 1).append("\n");
}
System.out.print(sb.toString());

输出结果：

Line 1
Line 2
Line 3
Line 4
Line 5

注意事项

  • 使用 println() 和 print() 方法时，Java 会自动调用对象的 toString() 方法。
  • 对于大量的输出操作，考虑使用缓冲输出流（如 BufferedWriter）以提高性能。
  • 在处理文件或网络输出时，记得正确处理可能出现的 IOException。
